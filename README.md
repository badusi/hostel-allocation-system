# Hostel Allocation System

An automated web-based application designed to manage and streamline the process of assigning hostel rooms to students in educational institutions. This system simplifies room applications, tracks room availability in real-time, balances student distribution, and eliminates manual paperwork for hostel administrators.

## 🚀 Features
* **Student Registration & Portal:** Students can view available hostels, check room fees, and apply for accommodation online.
* **Real-time Room Availability:** Automatic updates on available spaces, occupied rooms, and fully booked hostel blocks.
* **Automated Allocation:** Smart room assignment based on predefined criteria (e.g., year of study, department, or first-come, first-served).
* **Admin Management Dashboard:** Complete control for administrators to manage hostel blocks, set room capacities, view student details, and approve allocations.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL (via XAMPP phpMyAdmin)

## 💻 How to Run the Project
1. Download or clone this repository as a `.zip` file.
2. Extract the folder and move it into your XAMPP installation directory: `C:\xampp\htdocs\`
3. Open the **XAMPP Control Panel** and start both **Apache** and **MySQL**.
4. Open your browser and go to `http://localhost/phpmyadmin/` to create a new database.
5. Import your project's `.sql` database file (if included) into phpMyAdmin.
6. Open a new tab in your browser and visit: `http://localhost/hostel-allocation-system/`
